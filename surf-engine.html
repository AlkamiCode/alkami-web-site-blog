Surf Engine

Surf Engine is the worlds first search engine for North American surfing spots, some of the features of this app:

  • User login with Facebook
  • Google Maps Integration
  • Social Media sharing
  • Filter surf spots by experience level or spot name

Technical (Geek-Out!) Specs

  • Ruby on Rails
  • JavaScript, JQuery, Twitter Bootstrap on the Front-End
  • All app routing is Restful
  • Unit tested using RSpec, and feature tested using Capybara
  • Real time search filters
