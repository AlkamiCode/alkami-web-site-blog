Farmers Market

Farmers Market is a multi-tenant e-commerce platform that can accommodate multiple small business. This app is responsive across all screen sizes. Key features of this app:

  • Custom user authentication.
  • Supports multiple user types including Customer and Farm Owner
  • Store Admin authorization with custom features including adding/removing/hiding categories and items. Admin may also view all user orders and their status.
  • Customer Dashboard with past purchase history
  • Email notification upon new user registration and upon order completion.
  • Payment processing with Stripe(PayPals little brother).
  • Simple responsive design, feels great on mobile, minimum click-to-order process.
  • Full cart functionality allowing a user to update item quantity.

Technical (Geek-Out!) Specs

  • Ruby on Rails
  • JavaScript, JQuery, Bootstrap/Bootswatch on the Front-End
  • AWS S3 Cloud Storage for platform image storage
  • Twillio Messaging Capability
  • All app routing is Restful
  • Over 95% test coverage measured by SimpleCov
  • Unit tested using RSpec, and feature tested using Capybara
